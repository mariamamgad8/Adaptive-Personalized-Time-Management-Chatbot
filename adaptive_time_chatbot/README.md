# Adaptive Personalized Time Management Chatbot

A chatbot-based web application that helps students organize their time based on tasks, deadlines, routine, fixed commitments, and disruptions.

## Features
- Weekly schedule generation
- ML-based time block prediction
- Missed task recovery
- Wasted time replanning
- New event handling
- Streamlit chatbot interface

## Project Structure
- app/: Streamlit frontend
- chatbot/: parser and chatbot engine
- scheduling/: scheduler and replanner
- models/: trained model and encoders
- data/: CSV files
- utils/: helper functions

## Run
streamlit run app/streamlit_app.py
